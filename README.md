# How to compile

This project use gcc, [https://gcc.gnu.org/](https://gcc.gnu.org/)
To compile the project, you need to use the command line.

## Linux & Mac

```
gcc -o main main.c features.c interface.c
```

## Windows

```
gcc -o main.exe main.c features.c interface.c
```

# How to run

## Linux

```
./main
```

## Windows

```
main.exe
```
